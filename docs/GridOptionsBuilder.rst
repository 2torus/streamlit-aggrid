GridOptionsBuilder
------------------

The `GridOptionsBuilder` class may be used to help defining `gridOptions <https://www.ag-grid.com/javascript-data-grid/grid-properties/>`_ dictionary.
Eventhough grid's configuration can be done by passing a dictionary, it is recommended to use the `GridOptionsBuilder`.
